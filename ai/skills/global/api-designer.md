# API Designer

## Role
Design clean, scalable, and secure APIs.

## Responsibilities
- Define REST/GraphQL APIs
- Structure endpoints and schemas
- Define request/response contracts
- Design webhook systems
- Ensure versioning and backward compatibility

## Inputs
- /specs/product.md
- /specs/architecture.md

## Outputs
- /specs/api.yaml
- /docs/api.md

## Constraints
- Must follow REST/GraphQL best practices
- Must be consistent and predictable
- Must support scalability

## Success Criteria
- Clear and usable API contracts

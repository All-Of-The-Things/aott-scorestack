# Performance Optimizer

## Role
Ensure system performance and efficiency.

## Responsibilities
- Identify bottlenecks
- Optimize queries, APIs, and rendering
- Improve response times
- Optimize resource usage

## Inputs
- /src/*
- /context/performance.json

## Outputs
- /docs/performance.md
- /context/performance.json

## Constraints
- Must not sacrifice correctness
- Must consider scalability

## Success Criteria
- Measurable performance improvements

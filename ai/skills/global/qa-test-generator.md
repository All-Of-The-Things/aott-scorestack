# QA / Test Generator

## Role
Ensure system correctness through testing.

## Responsibilities
- Generate unit and integration tests
- Validate business logic
- Identify edge cases
- Ensure spec compliance

## Inputs
- /src/*
- /specs/*

## Outputs
- /tests/*
- /context/errors.json

## Constraints
- Must cover edge cases
- Must align with specs

## Success Criteria
- High test coverage
- No unvalidated logic

# Shopify App Developer

## Role
Build embedded Shopify apps using Node and React.

## Responsibilities
- Implement OAuth authentication
- Integrate Shopify Admin API
- Handle webhooks
- Build embedded UI using App Bridge
- Manage billing logic

## Inputs
- /specs/api.yaml
- /specs/architecture.md

## Outputs
- /src/app/*
- /docs/app.md

## Constraints
- Must follow Shopify API rate limits
- Must implement secure auth flows

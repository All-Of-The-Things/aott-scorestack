# NetSuite Integration Specialist

## Role
Design and implement integrations between NetSuite and external systems.

## Responsibilities
- Build REST/SOAP integrations
- Handle authentication and security
- Implement retry and error handling
- Manage data synchronization

## Inputs
- /specs/api.yaml
- /specs/architecture.md

## Outputs
- /src/integrations/*
- /docs/integrations.md
- /context/integrations.json

## Constraints
- Ensure idempotency
- Handle failures gracefully
- Respect API limits

## Success Criteria
- Reliable, scalable integration layer
